* 問題
  * ローマ数字を整数に変換する
* ローマ数字の値
  * I=1, V=5, X=10, L=50, C=100, D=500, M=1000
* ルール
  * 通常は左から右へ大きい順に並び、各値を足す
  * 小さい値が大きい値の前にある場合は引き算
    * IV=4, IX=9 (Iが前)
    * XL=40, XC=90 (Xが前)
    * CD=400, CM=900 (Cが前)
* 制約
  * 1 <= s.length <= 15
  * s は 'I', 'V', 'X', 'L', 'C', 'D', 'M' のみ
  * 範囲: [1, 3999]
* 例
  * "III" → 3 (1+1+1)
  * "LVIII" → 58 (50+5+1+1+1)
  * "MCMXCIV" → 1994 (1000+900+90+4)
