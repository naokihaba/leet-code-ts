* 問題
  * 配列内の2つの数値を足して target になるインデックスを返す
* 条件
  * 各入力には必ず1つの解が存在する
  * 同じ要素を2回使用できない
  * 答えはどの順序で返しても良い
* 制約
  * 2 <= nums.length <= 10^4
  * -10^9 <= nums[i] <= 10^9
  * -10^9 <= target <= 10^9
* 例
  * nums=[2,7,11,15], target=9 → [0,1]
  * nums=[3,2,4], target=6 → [1,2]
  * nums=[3,3], target=6 → [0,1]
* 発展課題
  * O(n^2) 未満の計算量で解けるか？
